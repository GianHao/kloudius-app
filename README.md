## Overview

This is a simple authentication demo built with Expo + React Native + TypeScript.
It demonstrates how to implement login, signup, persistent authentication, and protected navigation using modern React patterns.

## Demo Video
https://github.com/user-attachments/assets/b5e19831-d2eb-46bf-9f5a-dcf5a2ec77ea

## Features

- Signup with name, email, and password
- Login using stored credentials
- Logout functionality
- Persistent login using AsyncStorage
- React Context API for global auth state
- Stores multiple users locally

## Setup Instructions

1. Clone the project
   
   ````
   git clone https://github.com/GianHao/kloudius-app.git
   cd kloudius-app
   ````
   
2. Install dependencies
   
   ````
   npm install
   ````
   
3. Start the app
   
   ````
   npx expo start
   ````

## Tech Stack

- React Native

- TypeScript
- React Navigation
